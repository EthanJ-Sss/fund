# 基金购买与筛选快速指南

> 快速上手基金投资的实用指南

---

## 一、基金购买渠道

### 🟢 入门级（最简单）

适合：零基础、随时随地想操作的用户

| 渠道 | 入口 | 费率 | 优点 | 缺点 |
|-----|------|-----|------|-----|
| **支付宝（蚂蚁财富）** | 支付宝 App → 理财 → 基金 | 申购费1折 | 使用最方便、基金全、支持定投 | 部分基金独占期会限制转出 |
| **微信（理财通）** | 微信 → 我 → 服务 → 理财通 | 申购费1折 | 微信内操作、入口熟悉 | 基金数量略少于支付宝 |
| **银行 App** | 各银行手机App理财专区 | 一般不打折 | 可看到银行专属产品 | 费率高、基金选择有限 |

**推荐**：首选 **支付宝** 或 **微信**，3分钟完成第一笔投资

---

### 🟡 进阶级（需要开户）

适合：有一定投资经验、追求更多功能的用户

| 渠道 | 开户方式 | 费率 | 优点 | 缺点 |
|-----|---------|-----|------|-----|
| **天天基金** | App注册 + 绑卡 | 申购费1折 | 基金最全、筛选工具强大、专业数据 | 需单独安装App |
| **蛋卷基金** | App注册 + 绑卡 | 申购费1折 | 指数估值好用、有投顾组合 | 基金数量一般 |
| **且慢** | App注册 + 绑卡 | 申购费1折 | 智能投顾、懒人策略 | 更适合组合投资 |

**推荐**：想认真研究基金的用 **天天基金**，想省心的用 **且慢**

---

### 🔴 专业级（需证券开户）

适合：有股票账户、追求交易灵活性的用户

| 渠道 | 开户方式 | 费率 | 优点 | 缺点 |
|-----|---------|-----|------|-----|
| **券商App** | 线上开户（需身份证+银行卡+视频认证） | 场内ETF免申购费，仅收交易佣金(万1-万3) | 可买场内ETF、交易灵活、T+0到账 | 开户流程较长 |
| **基金公司直销** | 各基金公司官网/App | 部分免申购费 | 费率最低 | 只能买该公司产品、管理麻烦 |

**推荐**：有股票账户的，ETF用 **券商**买；长期持有单一基金公司产品可考虑 **直销**

---

### 渠道选择速查表

| 你的情况 | 推荐渠道 | 理由 |
|---------|---------|-----|
| 刚接触基金，想试试 | 支付宝/微信 | 最方便，无额外操作 |
| 想每月定投 | 支付宝/天天基金 | 定投功能完善，费率低 |
| 想深入研究筛选基金 | 天天基金 | 数据最全，筛选工具专业 |
| 想买ETF、频繁交易 | 券商 | 交易成本最低，实时成交 |
| 只买某家公司的基金 | 基金公司直销 | 费率最优 |
| 什么都不想管 | 且慢/蛋卷 | 智能投顾帮你搭配 |

## 二、基金筛选逻辑

### 筛选三步走

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: 选类型   →   Step 2: 看指标   →   Step 3: 比细节  │
│  (决定风险收益)       (淘汰差的)           (挑出最优)       │
└─────────────────────────────────────────────────────────┘
```

---

### Step 1: 选类型（先定方向）

| 资金用途 | 推荐类型 | 预期收益 | 风险等级 |
|---------|---------|---------|---------|
| 随时要用的钱 | 货币基金 | 1.5%-2% | ⭐ |
| 1年内要用 | 短债基金 | 2%-4% | ⭐⭐ |
| 1-3年不用 | 债券基金/混合基金 | 3%-8% | ⭐⭐⭐ |
| 3年以上不用 | 指数基金/股票基金 | 8%-15% | ⭐⭐⭐⭐ |

**核心原则**：时间越长，可承受风险越高，预期收益越高

---

### Step 2: 看指标（必看4项）

#### ① 看业绩（最重要）

```
✅ 近3年收益：排名前30%
✅ 近5年收益：排名前30%
❌ 只看近1个月/近1年（短期波动大，不靠谱）
```

#### ② 看回撤（亏损承受力）

```
最大回撤 = 历史上最大亏损幅度

参考标准：
- 债券基金：< 5%
- 混合基金：< 20%
- 股票基金：< 30%

❗ 选择自己能承受的回撤范围
```

#### ③ 看规模

```
✅ 2亿 - 100亿（最佳区间）
❌ < 1亿（可能清盘）
❌ > 200亿（船大难掉头，收益可能下降）
```

#### ④ 看基金经理

```
✅ 任职 > 3年
✅ 经历过牛熊周期
✅ 管理该基金期间业绩稳定
❌ 新手经理、频繁跳槽
```

---

### Step 3: 比细节（挑最优）

| 对比项 | 好的标准 | 说明 |
|-------|---------|-----|
| 费率 | 管理费 < 1.5%，托管费 < 0.25% | 长期持有费率影响大 |
| 夏普比率 | > 1 | 风险收益性价比高 |
| 基金公司 | 前20名大公司 | 投研实力强 |
| 持仓集中度 | 前10大持仓 < 60% | 过度集中风险大 |

---

### 4433法则（经典筛选法）

```
4：近1年排名前 1/4
4：近2年、3年、5年排名均前 1/4
3：近6个月排名前 1/3
3：近3个月排名前 1/3

→ 同时满足 = 长期短期都优秀
```

**使用方法**：天天基金/晨星网等平台可直接设置筛选

---

### 不同类型基金的筛选重点

#### 指数基金（被动型）

```
1. 跟踪误差 < 2%（越小越好）
2. 费率越低越好
3. 规模 > 1亿
4. 选头部基金公司的产品
```

**推荐指数**：
- 入门：沪深300、中证500
- 稳健：中证红利
- 进取：创业板指

#### 主动基金（需选基金经理）

```
1. 基金经理任职 > 3年
2. 近3年排名前30%
3. 最大回撤可接受
4. 规模 5-80亿
5. 投资风格稳定（不漂移）
```

## 三、新手行动建议

### 第一步：开始定投

```
1. 打开支付宝 → 基金
2. 搜索「沪深300」或「中证500」
3. 选择规模大、费率低的ETF联接基金
4. 设置每周/每月定投，金额量力而行
5. 坚持1年以上
```

### 定投金额参考

```
月收入的 10%-30% 用于定投

示例：
月入 1万 → 每月定投 1000-3000元
月入 2万 → 每月定投 2000-6000元
```

### 记住三不要

```
❌ 不要追涨杀跌（跌了更应该买）
❌ 不要频繁操作（手续费吃收益）
❌ 不要投入急用钱（至少3年不用的钱）
```
